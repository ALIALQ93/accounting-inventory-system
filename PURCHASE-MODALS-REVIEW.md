**مثال من الكود الحالي:**
```javascript
openSupplierPicker() {
    const modalElement = document.getElementById('supplierPickerModal');
    if (!modalElement) {
        console.error('❌ supplierPickerModal not found in DOM');
        return;
    }
    
    // Check if modal instance already exists
    let modal = bootstrap.Modal.getInstance(modalElement);
    if (!modal) {
        modal = new bootstrap.Modal(modalElement);
    }
    
    this.populateSupplierPicker();
    modal.show();
}
```

**النتيجة:** ✅ تم حل المشكلة - الكود الآن آمن ولا يسبب أخطاء عند عدم وجود العناصر
