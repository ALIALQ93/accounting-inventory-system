# 🏢 نظام إدارة مراكز الكلف - Cost Centers Management

## 📋 نظرة عامة

تم تطوير نظام احترافي ومتكامل لإدارة مراكز الكلف (Cost Centers) في النظام المحاسبي، مما يتيح تتبع دقيق للتكاليف والإيرادات عبر الأقسام والفروع والمشاريع المختلفة.

---

## ✨ الميزات الرئيسية

### **1️⃣ إدارة كاملة لمراكز الكلف**
- ✅ إضافة مراكز كلف جديدة
- ✅ تعديل معلومات المراكز الموجودة
- ✅ حذف المراكز غير المستخدمة
- ✅ البحث والتصفية السريعة

### **2️⃣ أنواع متعددة من المراكز**
```
🏢 قسم (Department)
   • قسم المبيعات
   • قسم الإنتاج
   • قسم الإدارة
   • قسم الموارد البشرية

📍 فرع (Branch)
   • فرع بغداد
   • فرع البصرة
   • فرع أربيل

📊 مشروع (Project)
   • مشروع التوسع
   • مشروع الصيانة
   • مشروع التطوير

📦 أخرى (Other)
   • أي تصنيف آخر حسب الحاجة
```

### **3️⃣ بطاقة معلومات شاملة**
```javascript
{
  code: "CC001",           // كود فريد (يُنشأ تلقائياً)
  name: "قسم المبيعات",    // اسم المركز
  type: "department",      // نوع المركز
  manager: "أحمد محمد",    // اسم المسؤول (اختياري)
  status: "active",        // الحالة (نشط/غير نشط)
  description: "...",      // وصف تفصيلي
  createdAt: timestamp,
  updatedAt: timestamp
}
```

### **4️⃣ إحصائيات تفاعلية**
```
┌──────────────────────────────────────┐
│ 📊 إجمالي المراكز: 15               │
│ ✅ المراكز النشطة: 12               │
│ 🏢 الأقسام: 8                       │
│ 📊 المشاريع: 4                      │
└──────────────────────────────────────┘
```

### **5️⃣ بحث وتصفية متقدمة**
- 🔍 البحث بالاسم أو الكود أو اسم المسؤول
- 🏷️ تصفية حسب النوع (قسم/فرع/مشروع/أخرى)
- 🔄 تصفية حسب الحالة (نشط/غير نشط)
- 🗑️ مسح جميع الفلاتر بنقرة واحدة

## 🚀 الاستخدام

### **1. الوصول للنظام**
```
النافذة الجانبية → المحاسبة → مراكز الكلفة
```

### **2. إضافة مركز كلفة جديد**
```
1. اضغط "إضافة مركز كلفة"
2. املأ البيانات:
   • الكود: CC001 (تلقائي)
   • الاسم: قسم المبيعات
   • النوع: قسم
   • المسؤول: أحمد محمد (اختياري)
   • الحالة: نشط
   • الوصف: مسؤول عن كافة عمليات المبيعات
3. احفظ ✓
```

### **3. التعديل والحذف**
```
• تعديل: اضغط ✏️ → عدّل البيانات → احفظ
• حذف: اضغط 🗑️ → تأكيد الحذف
```

### **4. البحث والتصفية**
```
• ابحث بالاسم: "قسم"
• اختر النوع: "قسم"
• اختر الحالة: "نشط"
```

## 🔗 الربط مع الأنظمة الأخرى

### **سيتم ربطها مع:**
```
✅ فواتير المبيعات
   → تسجيل كل فاتورة على مركز كلفة محدد

✅ فواتير المشتريات
   → تخصيص المشتريات للمراكز

✅ القيود اليومية
   → كل قيد يُسجل على مركز كلفة

✅ التقارير المالية
   → تقارير مفصلة لكل مركز:
     • إجمالي الإيرادات
     • إجمالي المصروفات
     • صافي الربح/الخسارة
```
